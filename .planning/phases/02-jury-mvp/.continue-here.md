## Critical Anti-Patterns (do NOT repeat these)
- **SILENT LLM FALLBACK**: If Qwen credentials missing → `gated_skip` → `RuntimeError` → stop. NEVER add keyword matching or rule-based substitutes.
- **NARROW GEO NEEDLES**: When adding a new data source with Russian/non-English column names, always extend `_find_column` needle lists immediately.
- **512 max_tokens FOR RUSSIAN**: Russian prose compresses poorly; always use ≥1024 for structured outputs, ≥2048 for narrative generation.
